This is documentation of a calculator I'm making purely from logic gates (aside from data converters for visual outputs). The goal is to get a better understanding of boolean algebra and low level logic, all .circ files were simulated in logisim evolution, brought into easyEDA to be turned into schematics, and finally built physically via breadboard. In the circut I will be using the 74HCxx series of IC's, a CMOS IC series, for their dynamic power range and low cost.
